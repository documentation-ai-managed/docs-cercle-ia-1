## Overview

Cercle IA prioritizes your data privacy by design. You can enable automatic anonymization, configure Private GPT for local model hosting, and manage sharing preferences directly from your dashboard. These features ensure that sensitive information stays secure, especially when using multiple AI models.

<Callout kind="info">
  Cercle IA never shares your prompts or responses with third-party providers unless explicitly configured. Private GPT runs models on isolated servers for maximum confidentiality.
</Callout>

## Enable Automatic Data Anonymization

Automatic anonymization scrubs personally identifiable information (PII) from your inputs before sending them to external AI models. This feature is off by default.

<Steps>
  <Step title="Access Settings" icon="settings">
    Navigate to your dashboard at `https://dashboard.example.com/settings/privacy`.
  </Step>
  <Step title="Toggle Anonymization" icon="shield">
    Find the `Enable Automatic Anonymization` switch and set it to on.
  </Step>
  <Step title="Verify Configuration" icon="check-circle">
    Test with a sample prompt containing mock PII like `john.doe@example.com`.
  </Step>
</Steps>

<ParamField path="privacy.anonymize" param-type="boolean" required="false" default="false">
  Automatically remove PII such as emails, names, and phone numbers from prompts.
</ParamField>

<ParamField path="privacy.redact_patterns" param-type="array" required="false">
  Custom regex patterns for additional redaction, e.g., `["\\b[A-Z0-9._%+-]+@[A-Z0-9.-]+\\.[A-Z]{2,}\\b"]`.
</ParamField>

## Configure Private GPT for Local Hosting

Private GPT hosts models locally on Cercle IA servers, ensuring no data leaves your controlled environment. Use it for confidential tasks.

<Tabs>
  <Tab title="Dashboard Setup" icon="monitor">
    <Steps>
      <Step title="Select Private GPT" icon="lock">
        In the model selector, choose `Private GPT` from the dropdown.
      </Step>
      <Step title="Upload Custom Model" icon="upload">
        Drag and drop your fine-tuned model file (GGUF format recommended).
      </Step>
    </Steps>
  </Tab>
  <Tab title="API Configuration" icon="code">
    Configure via API for programmatic access.

````javascript
const config = {
  model: "private-gpt-local",
  privacy: {
    localOnly: true,
    maxTokens: 4096
  }
};

const response = await fetch("https://api.example.com/v1/chat/completions", {
  method: "POST",
  headers: { "Authorization": `Bearer ${YOUR_API_KEY}` },
  body: JSON.stringify({ ...config, messages: [...] })
});
````
  </Tab>
</Tabs>

<CodeGroup tabs="JavaScript,Python">
````javascript
// Enable Private GPT in client
const cercleIA = new CercleIA({ apiKey: 'YOUR_API_KEY' });
cercleIA.setPrivacyMode('private-gpt');
````

````python
# Enable Private GPT in Python SDK
import cercle_ia

client = cercle_ia.CercleIA(api_key="YOUR_API_KEY")
client.config.privacy_mode = "private-gpt"
````
</CodeGroup>

## Manage Data Sharing Preferences

Control what data Cercle IA stores or shares.

| Preference | Description | Default |
|------------|-------------|---------|
| Store Conversations | Keep chat history for 30 days | On |
| Share with Models | Send to external providers | Off for Private GPT |
| Analytics Opt-out | Exclude from usage stats | Off |

<ParamField query="store_conversations" param-type="boolean" required="false" default="true">
  Toggle storage of conversation history.
</ParamField>

<ParamField query="share_analytics" param-type="boolean" required="false" default="false">
  Allow anonymized data for product improvements.
</ParamField>

## Best Practices for Confidential Information

Follow these guidelines to maximize security.

<Columns cols={2}>
  <Card title="Use Private GPT" icon="shield" horizontal>
    For sensitive data, always select Private GPT models. No external transmission occurs.
  </Card>
  <Card title="Review Logs" icon="file-text">
    Regularly check audit logs in `https://dashboard.example.com/logs`.
  </Card>
  <Card title="Custom Redaction" icon="edit-3">
    Define `redact_patterns` for domain-specific PII.
  </Card>
  <Card title="API Keys Securely" icon="key">
    Rotate keys monthly and use environment variables.
  </Card>
</Columns>
