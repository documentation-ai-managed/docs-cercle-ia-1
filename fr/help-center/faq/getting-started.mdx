## Vue d’ensemble

Créer un nouveau projet de documentation dans Cercle IA est simple.

## Étapes

1. Connectez-vous à votre tableau de bord Cercle IA.
2. Cliquez sur **"Nouveau projet"** depuis la page d’accueil.
3. Saisissez le nom de votre projet et une description facultative.
4. Cliquez sur **"Créer"**.
5. Votre projet apparaît dans la liste de l’espace de travail.

**Astuce :** Organisez vos documents dans des dossiers pour plus de clarté.
