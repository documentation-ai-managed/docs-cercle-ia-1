## Restez à jour

<Callout kind="info" title="Abonnez-vous aux mises à jour">

Suivez ce journal des modifications pour être averti des nouvelles intégrations de modèles d’IA, des améliorations de performance et des corrections de bugs. Vous recevez des notifications par email pour les versions majeures lorsque vous êtes inscrit dans les paramètres de votre compte sur `https://app.cercle-gpt.com`.

</Callout>

Cercle IA est mis à jour fréquemment pour intégrer les derniers modèles d’IA, renforcer les fonctionnalités de confidentialité et améliorer l’expérience utilisateur. Consultez les versions récentes ci-dessous pour voir ce qui change.

<Update label="2024-10-20" description="v1.3.0" tags={["feature", "improvement"]}>

## Nouvelles fonctionnalités

- Intégration de Perplexity AI pour des requêtes de recherche avancées
- Ajout de l’hébergement Private GPT pour le traitement de données confidentielles
- Introduction d’une vue de comparaison jusqu’à 5 modèles d’IA côte à côte

## Améliorations

- Réduction de la latence de 30 % pour les modèles GPT-5 mini et nano
- Amélioration de l’interface pour le changement de modèle avec bascule en un clic

## Corrections de bugs

- Correction de problèmes de chargement intermittents dans les conversations multi-modèles
- Résolution des erreurs de limite de jetons dans les sessions à long contexte

</Update>

<Update label="2024-10-05" description="v1.2.0" tags={["feature", "bugfix"]}>

## Nouvelles fonctionnalités

- Ajout des modèles Mistral pour la génération de code et le support multilingue
- Accès bêta aux entrées multimodales (analyse d’images avec Gemini)

## Améliorations

- Optimisation de la consommation de crédits pour l’usage de Private GPT
- Meilleure gestion des erreurs pour les appels API soumis à des limites de débit

## Corrections de bugs

- Correction des problèmes de synchronisation entre l’historique de conversation et les comparaisons de modèles
- Correction de la réactivité mobile dans le panneau de sélection de modèles

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "breaking"]}>

## Nouvelles fonctionnalités

- Moteur principal de comparaison multi-IA avec ChatGPT, Claude et Gemini
- Tableau de bord utilisateur pour suivre l’usage sur plus de 20 modèles

## Changements majeurs

- Mise à jour du flux d’authentification pour utiliser exclusivement le format `YOUR_API_KEY`

## Corrections de bugs

- Correction des boucles de redirection lors de la connexion initiale
- Stabilisation de la gestion de session pour les requêtes de longue durée

</Update>

## Comment mettre à jour

<Steps>
  <Step title="Vérifier la version" icon="check-circle">

    Connectez-vous à `https://app.cercle-gpt.com` et consultez votre version actuelle dans les paramètres.

  </Step>
  <Step title="Activer les mises à jour automatiques" icon="settings">

    Activez les mises à jour automatiques dans les préférences de votre compte pour recevoir instantanément les nouveaux modèles.

  </Step>
  <Step title="Signaler un problème" icon="alert-triangle">

    Utilisez le formulaire de retour d’expérience ou contactez le support si vous rencontrez des problèmes.

  </Step>
</Steps>

<Callout kind="tip" title="Vos retours comptent">

Partagez vos avis sur les nouvelles fonctionnalités ou les bugs via le bouton de feedback dans l’application. Vos retours orientent les prochaines versions.

</Callout>
